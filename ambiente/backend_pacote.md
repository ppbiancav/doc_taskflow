<h1>Como rodar a API localmente</h1>

Para rodar o projeto, é necessário ter instalado no computador as seguintes dependências:

> **Node.js**
>
> **npm**: npm install
>
> **MongoDB**

Vai ser necessário também configurar as váriaveis de ambiente já que são dados pessoais que não podem ser compartilhados por aqui.

Depois de clonar o projeto e instalar o mongoDB, no seu .env é importante que tenha a URL local de quem criou no banco de dados no projeto , algo como:

> **DATABASE_URL=**
>

# Nesse caso irei passar a URL criada

E para rodar, apenas digite **npm run dev** no terminal do bash.

A API estará disponível em: http://localhost:8000
