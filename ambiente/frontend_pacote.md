# para rodar o frontend do Task Flow localmente..

<h1>Como rodar o frontend localmente</h1>

Para rodar o projeto, é praticamente o mesmo esquema do backend, a diferença é que não haverá necessidade de incrementar banco de dados no processo:

> **Node.js**
>
> **npm**: npm install

E para rodar, apenas digite **npm run dev** ou **npm start** no terminal do bash.

A API estará disponível em: http://localhost:5173
